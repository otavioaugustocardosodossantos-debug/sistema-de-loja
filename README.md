# sistema-de-loja
Sistema gerenciador de atividades e produtos para lojas físicas e digitais.




Sistema Gerenciador de Atividades e Produtos

Projeto Integrado – Programação Orientada a Objetos com Banco de Dados

1. Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema gerenciador de atividades e produtos voltado para lojas físicas e digitais. A aplicação foi construída utilizando os conceitos de Programação Orientada a Objetos (POO) integrados a um banco de dados relacional, permitindo armazenamento persistente, organização e manipulação eficiente das informações.

O sistema tem como objetivo auxiliar na gestão operacional do negócio, organizando produtos, tarefas administrativas e processos internos de forma estruturada e automatizada.

2. Objetivos
Objetivo Geral

Desenvolver uma aplicação orientada a objetos integrada a banco de dados para gerenciamento de produtos e atividades comerciais.

Objetivos Específicos

Aplicar os princípios da Programação Orientada a Objetos:

Encapsulamento

Abstração

Herança

Polimorfismo

Implementar persistência de dados utilizando banco de dados relacional

Criar estrutura organizada e escalável

Simular um ambiente real de gestão comercial

3. Problema Resolvido

Lojas físicas e digitais enfrentam dificuldades na organização simultânea de:

Cadastro e controle de produtos

Atualização de estoque

Registro de atividades internas

Organização de tarefas administrativas

Este sistema propõe uma solução integrada para centralizar essas funcionalidades em uma única aplicação.

5. Estrutura do Projeto

O sistema é dividido em camadas:

Camada de Modelo (Entidades)

Camada de Controle (Regras de Negócio)

Camada de Persistência (Banco de Dados)

Camada de Interface (Console ou Web, dependendo da implementação)

Essa organização segue princípios de boas práticas e separação de responsabilidades.

Este projeto integra teoria e prática ao aplicar conceitos fundamentais de Programação Orientada a Objetos junto a banco de dados, simulando um sistema real de gestão comercial.
